# Plugin 兼容

该索引列出了完成 Rspack 兼容性测试的插件（Plugin）列表。

| Plugin                                                                                       | 兼容性   | 替代                                                                    | 备注                                                |
| -------------------------------------------------------------------------------------------- | -------- | ----------------------------------------------------------------------- | --------------------------------------------------- |
| [html-webpack-plugin](https://www.npmjs.com/package/html-webpack-plugin)                     | 有替代   | [builtins.html](/config/builtins#builtinshtml) 或者 @rspack/plugin-html | 目前并不支持 webpack 的 childCompiler 相关功能      |
| [DefinePlugin](https://webpack.js.org/plugins/define-plugin/)                                | 已内置   | [builtins.define](/config/builtins#builtinsdefine)                      |                                                     |
| [copy-webpack-plugin](https://www.npmjs.com/package/copy-webpack-plugin)                     | 部分兼容 | [builtins.copy](/config/builtinscopy)                                   | 暂不兼容最新版本，但兼容 copy-webpack-plugin@5 版本 |
| [mini-css-extract-plugin](https://www.npmjs.com/package/mini-css-extract-plugin)             | 已内置   | [type="css"](/guide/language-support#css)                               |                                                     |
| [terser-webpack-plugin](https://www.npmjs.com/package/terser-webpack-plugin)                 | 已内置   | [builtins.minifyOptions](/config/builtins#builtinsminifyoptions)        |                                                     |
| [progressPlugin](https://webpack.js.org/plugins/progress-plugin/)                            | 已内置   | [builtins.progress](/config/builtins#builtinsprogress)                  |                                                     |
| [webpack-bundle-analyzer](https://www.npmjs.com/package/webpack-bundle-analyzer)             | 已内置   |                                                                         | 已内置支持，--analyze 开启                          |
| [webpack-stats-plugin](https://www.npmjs.com/package/webpack-stats-plugin)                   | 兼容     |                                                                         |                                                     |
| [tsconfig-paths-webpack-plugin](https://www.npmjs.com/package/tsconfig-paths-webpack-plugin) | 已内置   | [resolve.tsConfigPath](/config/resolve#tsConfigPath)                    |
