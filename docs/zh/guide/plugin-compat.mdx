# 插件兼容

该索引列出了完成 Rspack 兼容性测试的插件 (Plugin) 列表。

| Plugin                  | 兼容性   | 替代                                                                    | 备注                                                         |
| ----------------------- | -------- | ----------------------------------------------------------------------- | ------------------------------------------------------------ |
| html-webpack-plugin     | 不兼容   | [builtins.html](/config/builtins#builtinshtml) 或者 @rspack/plugin-html | 目前并不支持 html-webpack-plugin 里的 childCompiler 相关功能 |
| DefinePlugin            | 不兼容   | [builtins.define](/config/builtins#builtinsdefine)                      |                                                              |
| copy-webpack-plugin     | 部分兼容 | [builtins.copy](/config/builtinscopy)                                   | 暂不兼容最新版本，但兼容 copy-webpack-plugin@5 版本          |
| mini-css-extract-plugin | 不兼容   | [type="css"](/guide/language-support#css)                               |                                                              |
| terser-webpack-plugin   | 不兼容   | [builtins.minifyOptions](/config/builtins#builtinsminifyoptions)        |                                                              |
| webpackbar              | 不兼容   | [builtins.progress](/config/builtins#builtinsprogress)                  |                                                              |
| webpack-bundle-analyzer | 兼容     |                                                                         | 已内置支持，--analyze 开启                                   |
| webpack-stats-plugin    | 兼容     |                                                                         |                                                              |
