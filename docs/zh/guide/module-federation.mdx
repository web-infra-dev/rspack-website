# 模块联邦

<ApiMeta addedVersion={'0.4.5'} stability={Stability.Experimental} />

模块联邦（Module Federation）允许 JavaScript 应用程序在运行时动态地加载来自其他构建的代码。这个功能使得各个独立构建的应用程序（微前端架构中的“微应用”）可以共享模块而无需重新编译整个应用。

Rspack 中支持了 Module Federation 1.0 和 Module Federation 1.5 两个版本：

- Module Federation 1.0 即当前 Webpack 实现的 ModuleFederationPlugin。
- Module Federation 1.5 在 1.0 的基础上增加了运行时插件功能，使其可以对 Module Federation 的行为和功能进行扩展。

详细配置可参考：

- [ModuleFederationPlugin](/config/plugins.html#containermodulefederationplugin)
- [ModuleFederationPluginV1](/config/plugins.html#containermodulefederationpluginv1)
