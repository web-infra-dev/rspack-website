# Vue

Vue 是一个非常流行的前端框架，Rspack 在未来计划实现完整的 Vue 支持，提供完整的工程化能力。

## vue-loader

Rspack 的目标是完整的兼容 vue-loader, 但是限于 Rspack 目前提供的 API 有限，目前无法直接运行 vue-loader, 但是你仍然可以自行构建 vue-loader 的功能， Rspack 提供一个简单的 vue-loader 实现供参考，你可以在这里看到完整实现 [vue-example](https://github.com/modern-js-dev/rspack/tree/main/examples/vue)。
