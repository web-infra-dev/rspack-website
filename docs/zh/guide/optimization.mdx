# 生产优化

## 包分析

Rspack CLI 支持使用`--analyze` 选项进行包分析。它在内部使用 [webpack-bundle-analyzer](https://www.npmjs.com/package/webpack-bundle-analyzer) 实现包分析功能。

```sh
$ Rspack build --analyze
```

你也可以生成 stats.json 文件，用其他包析工具进一步分析，如 [bundle-stats](https://github.com/relative-ci/bundle-stats) 或 [statoscope](https://statoscope.tech/)。

```sh
$ Rspack build --json stats.json
```

## 代码压缩

Rspack 默认使用内置的压缩器对 js 和 css 代码进行压缩，可以使用 [builtins.minifyOptions](config/builtins#builtinsminifyoptions) 来进行配置。若内置压缩器无法满足需求，可以使用 [optimization.minimizer](config/optimization#optimizationminimizer) 设置自定义压缩器。
