# 简介

Rspack 提供了多种接口来自定义编译过程，包括

## CLI

命令行界面(CLI), 用于配置构建过程并于之交互，支持了日常开发常用的 dev、build 等功能。详情可参考[CLI](/api/cli)

## Node API

Node API，用于在 Node.js 中使用 Rspack，。详情可参考[Node API](/api/node-api)

## Loader API

Loader 用于对模块的源码进行转换，详情可参考[Loader API](/api/loader-api)

## Plugin API

Plugin 允许用户介入编译流程。 插件可以在不同的生命周期上注册回调函数，从而修改用户的编译过程，详情可参考[Plugin API](/api/plugin-api)
