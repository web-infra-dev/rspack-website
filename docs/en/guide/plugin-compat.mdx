# Plugin compat

This index lists the plugins that have been verified by compatibility tests with Rspack.

| Plugin                                                                                       | Compatibility        | Alternative                                                                                                                | Remarks                                                                                                |
| -------------------------------------------------------------------------------------------- | -------------------- | -------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| [html-webpack-plugin](https://www.npmjs.com/package/html-webpack-plugin)                     | alternative          | [builtins.html](/config/builtins#builtinshtml) or [@rspack/plugin-html](https://www.npmjs.com/package/@rspack/plugin-html) | Currently does not support childCompiler related functionality in webpack                              |
| [DefinePlugin](https://webpack.js.org/plugins/define-plugin/)                                | included             | [builtins.define](/config/builtins#builtinsdefine)                                                                         |                                                                                                        |
| [copy-webpack-plugin](https://www.npmjs.com/package/copy-webpack-plugin)                     | partially compatible | [builtins.copy](/config/builtins.html#builtinscopy)                                                                        | Not currently compatible with the latest version, but is compatible with copy-webpack-plugin@5 version |
| [banner-plugin](https://webpack.js.org/plugins/banner-plugin)                                | partially compatible | [builtins.banner](/config/builtins.html#builtinsbanner)                                                                    | Not support function in banner                                                                         |
| [mini-css-extract-plugin](https://www.npmjs.com/package/mini-css-extract-plugin)             | included             | [type="css"](/guide/language-support#css)                                                                                  |                                                                                                        |
| [terser-webpack-plugin](https://www.npmjs.com/package/terser-webpack-plugin)                 | included             | [builtins.minifyOptions](/config/builtins#builtinsminifyoptions)                                                           |                                                                                                        |
| [progressPlugin](https://webpack.js.org/plugins/progress-plugin/)                            | included             | [builtins.progress](/config/builtins#builtinsprogress)                                                                     |                                                                                                        |
| [webpack-bundle-analyzer](https://www.npmjs.com/package/webpack-bundle-analyzer)             | included             |                                                                                                                            | built-in support, enabled with --analyze flag                                                          |
| [webpack-stats-plugin](https://www.npmjs.com/package/webpack-stats-plugin)                   | compatible           |                                                                                                                            |                                                                                                        |
| [tsconfig-paths-webpack-plugin](https://www.npmjs.com/package/tsconfig-paths-webpack-plugin) | included             | [resolve.tsConfigPath](/config/resolve#tsConfigPath)                                                                       |                                                                                                        |
| [HotModuleReplacementPlugin](https://webpack.js.org/plugins/hot-module-replacement-plugin/)  | compatible           | [devServe.hot](config/devtool#devserverhot)                                                                                |                                                                                                        |
